## ğŸ”¹ Data Preprocessing

1. **Drop rare classes:** Classes with fewer than 5 samples were removed to prevent errors during train/test split.
2. **Encode categorical features:**
   - `level` and `equipment` were one-hot encoded.
3. **Scale numeric features:**
   - `program_length` and `time_per_workout` were standardized using `StandardScaler`.
4. **Train/Test Split:**
   - 80% train, 20% test with stratification on `goal`.

---

## ğŸ”¹ Algorithm Used

- Easy to interpret and fast to train.
